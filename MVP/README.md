## ‚ö° Quick Start

To play this version, you must run a local server inside this folder to avoid browser security errors.

1.  Open your terminal/command prompt.
2.  Navigate to this **MVP** folder.
3.  Run the following command:
    ```bash
    http-server
    ```
4.  Open your browser and visit: `http://127.0.0.1:8080` (or the port shown in your terminal).

## üöÄ Current Features

In this version, the core pillars of the game are fully implemented and ready to play:

* **Tutorial Zone:** A safe environment to master the controls:
    * **WASD** for movement.
    * **Left Click** to attack.
* **The Village (NPC Zone):** Explore the starting area and interact with the world's inhabitants.
* **The Sage:** Speak to the village elder. His wisdom is generated via **Google Gemini AI** and changes dynamically over time to provide unique interactions.
* **Core Systems:**
    * **Inventory:** Manage your items.
    * **Economy:** Collect Lime Coins to ensure you can respawn.
    * **Combat Loop:** Basic hit detection and health management.

---

## üöß What's Missing? (Future Plans)

While the MVP sets the stage, the world is still growing. The following features are **not** in this build but are available in the **Full Version**:

* **Expanded Map:** The "Left" and "Right" enemy zones are currently inaccessible.
* **High-Level Combat:** There are fewer enemies here, serving mostly for testing and tutorial purposes.
* **Extended Story:** The deeper lore regarding the Lime-Blobs is reserved for the complete adventure.
